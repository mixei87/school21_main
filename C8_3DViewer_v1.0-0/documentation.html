Документация 3DViewer_V1.0

Программа 3DViewer_V1.0 предназначена для просмотра 3D моделей в каркасном виде. Сами модели загружаются из файлов формата .obj.

Формат файла .obj - это простой формат данных, который представляет только трехмерную геометрию, а именно положение каждой вершины, положение UV координат текстуры каждой вершины, нормали вершин и грани, которые определяют каждый многоугольник как список вершин и вершин текстуры. Координаты obj не имеют единиц измерения, но файлы obj могут содержать информацию о масштабе в удобочитаемой строке комментариев.

Базовые функции:

  • Перемещение модели на заданное расстояние относительно осей X, Y, Z.
  • Поворот модели на заданный угол относительно своих осей X, Y, Z
  • Масштабирование модели на заданное значение.

Дополнительные функции:

  • Программа позволяет настраивать тип проекции (параллельная и центральная)
  • Программа позволяет настраивать тип (сплошная, пунктирная), цвет и толщину ребер, способ отображения (отсутствует, круг, квадрат), цвет и размер вершин
  • Программа должна позволяет выбирать цвет фона
  • Настройки программы сохраняться между перезапусками
  • Программа позволяет сохранять полученные ("отрендеренные") изображения в файл в форматах bmp и jpeg
  • Программа позволяет по специальной кнопке записывать небольшие "скринкасты" - текущие пользовательские аффинные преобразования загруженного объекта в gif-анимацию (640x480, 10fps, 5s)
